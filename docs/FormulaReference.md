## 运算符

### 算术运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法（或字符串连接） | `=A1+B1` 或 `="Hello"+"World"` | 数值相加或字符串连接 |
| `-` | 减法 | `=A1-B1` | 数值相减 |
| `*` | 乘法 | `=A1*B1` | 数值相乘 |
| `/` | 除法 | `=A1/B1` | 数值相除 |
| `^` | 乘方 | `=2^3` | 8 |

### 比较运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `>` | 大于 | `=A1>10` |
| `<` | 小于 | `=A1<10` |
| `=` | 等于 | `=A1=10` |
| `>=` | 大于等于 | `=A1>=10` |
| `<=` | 小于等于 | `=A1<=10` |
| `<>` | 不等于 | `=A1<>10` |

### 逻辑运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `AND` | 逻辑与 | `=A1>0 AND B1>0` |
| `OR` | 逻辑或 | `=A1>0 OR B1>0` |

## 函数

### 数学函数

#### ABS(number)
返回数值的绝对值。

**参数：**
- `number` - 需要计算绝对值的数值

**示例：**
```
=ABS(-5)
```
**返回：** `5`

---

#### MOD(number1, number2)
返回两数相除的余数。

**参数：**
- `number1` - 被除数
- `number2` - 除数

**示例：**
```
=MOD(10, 3)
```
**返回：** `1`

---

#### POWER(base, exponent)
返回数的指定次幂。

**参数：**
- `base` - 底数
- `exponent` - 指数

**示例：**
```
=POWER(2, 3)
```
**返回：** `8`

---

#### SQRT(number)
返回数值的平方根。

**参数：**
- `number` - 需要计算平方根的数值

**示例：**
```
=SQRT(16)
```
**返回：** `4`

---

#### ROUND(number, decimals)
将数值四舍五入到指定的小数位数。

**参数：**
- `number` - 需要四舍五入的数值
- `decimals` - 小数位数

**示例：**
```
=ROUND(3.14159, 2)
```
**返回：** `3.14`

---

#### SIN(number)
返回角度的正弦值（弧度制）。

**参数：**
- `number` - 弧度值

**示例：**
```
=SIN(PI()/2)
```
**返回：** `1`

---

#### TAN(number)
返回角度的正切值（弧度制）。

**参数：**
- `number` - 弧度值

**示例：**
```
=TAN(0)
```
**返回：** `0`

---

#### PI()
返回圆周率 π 的值。

**示例：**
```
=PI()
```
**返回：** `3.141592653589793`

---

#### VALUE(string)
将字符串转换为数值。支持百分比格式。

**参数：**
- `string` - 需要转换的字符串

**示例：**
```
=VALUE("123")
=VALUE("50%")
```
**返回：** `123` 或 `0.5`

---

### 字符串函数

#### STRING(value)
将数值转换为字符串。

**参数：**
- `value` - 需要转换的数值

**示例：**
```
=STRING(123)
```
**返回：** `"123"`

---

#### STRLEN(string)
返回字符串的长度。

**参数：**
- `string` - 需要计算长度的字符串

**示例：**
```
=STRLEN("Hello")
```
**返回：** `5`

---

#### LEFT(string, length)
返回字符串左侧指定长度的字符。

**参数：**
- `string` - 源字符串
- `length` - 要提取的字符数

**示例：**
```
=LEFT("Hello World", 5)
```
**返回：** `"Hello"`

---

#### RIGHT(string, length)
返回字符串右侧指定长度的字符。

**参数：**
- `string` - 源字符串
- `length` - 要提取的字符数

**示例：**
```
=RIGHT("Hello World", 5)
```
**返回：** `"World"`

---

#### MID(string, start, length)
返回字符串中从指定位置开始的指定长度字符。

**参数：**
- `string` - 源字符串
- `start` - 起始位置（从0开始）
- `length` - 要提取的字符数

**示例：**
```
=MID("Hello World", 6, 5)
```
**返回：** `"World"`

---

#### FIND(string1, string2)
在字符串中查找子字符串的位置。

**参数：**
- `string1` - 被搜索的字符串
- `string2` - 要查找的子字符串

**示例：**
```
=FIND("Hello World", "World")
```
**返回：** `6`

---

#### UPPER(string)
将字符串转换为大写。

**参数：**
- `string` - 需要转换的字符串

**示例：**
```
=UPPER("hello")
```
**返回：** `"HELLO"`

---

#### LOWER(string)
将字符串转换为小写。

**参数：**
- `string` - 需要转换的字符串

**示例：**
```
=LOWER("HELLO")
```
**返回：** `"hello"`

---

#### TRIMLEFT(string)
去除字符串左侧的空白字符。

**参数：**
- `string` - 需要处理的字符串

**示例：**
```
=TRIMLEFT("   Hello")
```
**返回：** `"Hello"`

---

#### TRIMRIGHT(string)
去除字符串右侧的空白字符。

**参数：**
- `string` - 需要处理的字符串

**示例：**
```
=TRIMRIGHT("Hello   ")
```
**返回：** `"Hello"`

---

#### COMP(string1, string2)
比较两个字符串。

**参数：**
- `string1` - 第一个字符串
- `string2` - 第二个字符串

**返回值：**
- `-1` - string1 小于 string2
- `0` - string1 等于 string2
- `1` - string1 大于 string2

**示例：**
```
=COMP("Apple", "Banana")
```
**返回：** `-1`

### 日期函数

#### TODAY()
返回当前日期的时间戳。

**示例：**
```
=TODAY()
```
**返回：** 当前日期的时间戳（毫秒）

---

#### NOW()
返回当前日期和时间的时间戳。

**示例：**
```
=NOW()
```
**返回：** 当前日期时间的时间戳（毫秒）

---

#### DATE(year, month, day)
根据年、月、日创建日期的时间戳。

**参数：**
- `year` - 年份
- `month` - 月份（1-12）
- `day` - 日期（1-31）

**示例：**
```
=DATE(2024, 1, 1)
```
**返回：** 2024年1月1日的时间戳

---

#### DATEVALUE(dateString)
将日期字符串转换为时间戳。

**参数：**
- `dateString` - 日期字符串，支持"年"、"月"、"日"中文格式

**示例：**
```
=DATEVALUE("2024-01-01")
=DATEVALUE("2024年1月1日")
```
**返回：** 对应日期的时间戳

---

#### DATESTR(timestamp, style)
将时间戳转换为格式化的日期字符串。

**参数：**
- `timestamp` - 时间戳
- `style` - 格式样式

**样式值说明：**

| 样式值 | 格式说明 | 示例 |
|--------|----------|------|
| 0 | yyyy-M-d | 2024-1-1 |
| 1 | yyyy-MM-dd HH:mm:ss | 2024-01-01 12:00:00 |
| 8 | 中文大写 | 二〇二四年一月一日 |
| 9 | 中文年月 | 二〇二四年一月 |
| 10 | 中文月日 | 一月一日 |
| 11 | 中文简体 | 2024年1月1日 |
| 12 | 中文年月简体 | 2024年1月 |
| 13 | 中文月日简体 | 1月1日 |

**示例：**
```
=DATESTR(TODAY(), 0)
=DATESTR(TODAY(), 8)
```

### 条件函数

#### IF(condition, trueValue, falseValue)
根据条件返回不同的值。

**参数：**
- `condition` - 条件表达式
- `trueValue` - 条件为真时返回的值
- `falseValue` - 条件为假时返回的值

**示例：**
```
=IF(A1>10, "大于10", "小于等于10")
```

### 统计函数

#### SUM(range, condition)
计算单元格区域中满足条件的数值之和。

**参数：**
- `range` - 单元格区域（如 "A1:A10"）
- `condition` - 条件表达式，使用 `_loopcell_` 表示当前循环的单元格值

**示例：**
```
=SUM(A1:A10, True)
=SUM(A1:A10, _loopcell_>0)
```

---

#### AVERAGE(range, condition)
计算单元格区域中满足条件的数值的平均值。

**参数：**
- `range` - 单元格区域
- `condition` - 条件表达式

**示例：**
```
=AVERAGE(A1:A10, True)
=AVERAGE(A1:A10, _loopcell_>0)
```

---

#### MIN(range, condition)
返回单元格区域中满足条件的最小值。

**参数：**
- `range` - 单元格区域
- `condition` - 条件表达式

**示例：**
```
=MIN(A1:A10, True)
=MIN(A1:A10, _loopcell_>0)
```

---

#### MAX(range, condition)
返回单元格区域中满足条件的最大值。

**参数：**
- `range` - 单元格区域
- `condition` - 条件表达式

**示例：**
```
=MAX(A1:A10, True)
=MAX(A1:A10, _loopcell_>0)
```

---

#### STDEV(range)
计算单元格区域中数值的样本标准差。

**参数：**
- `range` - 单元格区域

**示例：**
```
=STDEV(A1:A10)
```

---

#### SELECT(range, condition, index)
从单元格区域中选择满足条件的第N个值。

**参数：**
- `range` - 单元格区域
- `condition` - 条件表达式
- `index` - 要选择的第几个值（从1开始）

**示例：**
```
=SELECT(A1:A10, _loopcell_>5, 1)
```
**返回：** 区域中第一个大于5的值

---

### 单元格函数

#### GETCELLVALUE(cellAddress)
获取指定单元格的值。

**参数：**
- `cellAddress` - 单元格地址（如 "A1"）

**示例：**
```
=GETCELLVALUE("A1")
```

---

#### LOOPCELL()
用于统计函数中，表示当前循环的单元格值。通常不单独使用，而是在条件表达式中使用。

**示例：**
```
=SUM(A1:A10, _loopcell_>0)
```

## 使用说明

### 公式语法

1. 公式以 `=` 开头
2. 函数名不区分大小写
3. 字符串使用双引号 `"` 包裹
4. 单元格引用使用列字母+行号格式（如 A1, B2, C3）
5. 区域引用使用冒号分隔（如 A1:B10）

### 运算优先级

1. 括号 `()`
2. 乘方 `^`
3. 乘除 `*` `/`
4. 加减 `+` `-`
5. 比较 `>` `<` `=` `>=` `<=` `<>`
6. 逻辑 `AND` `OR`

### 条件表达式

在统计函数（SUM、AVERAGE、MIN、MAX、SELECT）中，可以使用 `_loopcell_` 来表示当前循环的单元格值，用于构建条件表达式。

**示例：**
```
=SUM(A1:A10, _loopcell_>10)     // 求和所有大于10的值
=AVERAGE(A1:A10, _loopcell_<0)  // 求平均所有小于0的值
=SELECT(A1:A10, _loopcell_="张三", 1)  // 查找第一个值为"张三"的单元格
```

### 错误处理

当公式计算出错时，将返回 `#Error`。常见错误包括：
- 除数为零
- 函数参数类型错误
- 无效的单元格引用
- 未实现的函数
